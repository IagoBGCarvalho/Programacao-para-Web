Exemplo 7 - Tabelas em HTML

Quando precisamos criar tabelas no html, devemos usar a tag table juntamente com as tags th, tr e td

As tabelas são úteis para exibir dados de forma organizada e podem ser compostas por várias linhas e colunas. A seguir demonstramos de forma simples a criação de uma tabela em html.
