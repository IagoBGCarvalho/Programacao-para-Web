Web Semantica

A web semântica é uma abordagem que utiliza elementos html com significado claro para estruturar conteúdo na web. São exemplos de tags semânticas <header>, <nav>, <main>, <section>, <article>, <asside> e o <footer>.

Eu posso ter em um documento HTML várias seções.

Elemento semântico div

A tag html div é um elemento semântico com valor neutro. Ele é muito utilizado para organizar conteúdo e possui valor semântico neutro ou nulo

Seletores CSS

Os seletores css, são padrões usados para selecionar e estilizar elementos html. Junto com as propriedades e valores eles formam as regras css.

Seletor de Tipo

Os seletores de tipo, são aqueles que representam as tags html, por exemplo: h1 {propriedade:valor};

Seletor de Id

O seletor de Id é aquele que utiliza o atributo html id. Esse seletor é personalíssimo uma vez que em tese o elemento html só um id que pode se repetir. O seletor de Id deve começar com o #nomeSeletor{propriedades:valor};

Seletor de Classe

O seletor de Classe é aquele que é aplicado para estilizar um ou mais elementos html que possuam o mesmo atributo class. Esse seletor começa com um ponto seguido do nome do seletor. Por exemplo .nomeSeletor{propriedade:valor};

Tag semantica Aside

A tag semantica aside é aquela usada para representar conteúdo que está relacionado com o conteúdo principal, ou seja, é um conteúdo auxiliar. É um conteúdo extra. Geralmente é usado também como menu auxiliar,
