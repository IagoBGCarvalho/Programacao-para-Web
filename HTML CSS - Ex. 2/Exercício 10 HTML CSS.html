Vinculação interna - incorporada e inline

A vinculação inline é uma forma de aplicar estilos css diretamente a um elemento html específico, usando o atributo style. Diferentemente da vinculação externa, esse tipo de vinculação ocorre diretamente no elemento.

Esse tipo de vinculação é pouco recomendada pois dificulta a manutenção do documento html.

Nesta div estamos aplicando a vinculaão inline.
