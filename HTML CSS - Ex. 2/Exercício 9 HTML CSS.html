Exemplo 9 - HTML/CSS Vinculação de arquivos CSS

A vinculação de arquivos CSS pode ser feita de duas maneiras principais: interna e externa. Ambas permitem aplicar estilos a uma página web, mas diferem em como os estilos são definidos e organizados.

Vinculação Externa ou Linkada/Importada

A vinculação externa é aquela onde os estilos CSS são definidos em um arquivo separado (com extensão .css) e vinculados ao documento html por meio da tag link.

Neste exemplo usamos a vinculação externa, que é mais efetiva, pois com ela temos maior manutenibilidade e capacidade de reuso.

Vantagens da vinculação Externa

  • Separação de preocupaçõs - mantém o html focado no conteúdo e o css na apresentação
  • Facilidade de manutenção - Alterações nos estilos são feitas em um único arquivo.
  • Melhor desempenjo - o navegador pode armazenar em cache o arquivo css, reduzindo o tempo de carregamento
